# TaxSyncForDrivers-Mobile

This is the React Native (Expo) mobile app for TaxSyncForDrivers.

## Project Structure

```
TaxSyncForDrivers-Mobile/
├── src/
│   ├── components/
│   ├── screens/
│   ├── services/
│   ├── utils/
│   └── navigation/
├── docs/
├── __tests__/
├── .github/
└── README.md
```

## Getting Started

1. **Install Expo CLI (if not already):**
   ```sh
   npm install -g expo-cli
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Start the development server:**
   ```sh
   expo start
   ```
4. **Project folders:**
   - `src/components/` - Reusable UI components
   - `src/screens/` - App screens
   - `src/services/` - API and backend integration
   - `src/utils/` - Utility functions
   - `src/navigation/` - Navigation setup
   - `docs/` - Documentation
   - `__tests__/` - Tests
